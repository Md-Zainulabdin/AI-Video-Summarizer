## AI Video Summarizer

An advanced multimodal AI-powered video summarizer built using Streamlit and Gemini 2.0 Flash. This application allows users to analyze video content by uploading files or providing YouTube URLs, delivering detailed and actionable insights using AI and supplementary web research.
